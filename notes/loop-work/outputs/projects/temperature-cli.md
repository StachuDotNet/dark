# temperature-cli

**Goal:** Convert between Fahrenheit and Celsius via `to-c` and `to-f` subcommands, printing the result to 2 decimals.

**Kind:** greenfield

## Acceptance criteria
- [ ] `temperature-cli to-c <F>` converts Fahrenheit to Celsius, printed to 2 decimal places.
- [ ] `temperature-cli to-f <C>` converts Celsius to Fahrenheit, printed to 2 decimal places.
- [ ] At least four canonical conversions produce correct results.
- [ ] An unknown subcommand exits non-zero with usage.
- [ ] `temperature-cli --help` (or `-h`) prints usage and exits 0.
