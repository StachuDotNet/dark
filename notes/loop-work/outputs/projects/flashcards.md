# flashcards

**Goal:** Drill flashcards from a JSON deck and track attempts persistently.

**Kind:** greenfield

## Acceptance criteria
- [ ] Quizzes from a JSON deck and records attempt history that persists across runs.
- [ ] Matches golden-file output for fixture inputs.
- [ ] Unicode/emoji content renders without column misalignment.
- [ ] Empty input produces empty output and exits 0.
- [ ] Sorting is stable for equal keys.
- [ ] `--help` / `-h` prints usage and exits 0.
