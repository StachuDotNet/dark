# fizzbuzz

**Goal:** Print 1..N, replacing multiples of 3 with `Fizz`, 5 with `Buzz`, and 15 with `FizzBuzz`.

**Kind:** greenfield

## Acceptance criteria
- [ ] Reads N from a positional argument.
- [ ] `fizzbuzz 15` produces the canonical 15-line output (numbers, `Fizz`, `Buzz`, `FizzBuzz` in the right positions).
- [ ] Multiples of 3 print `Fizz`, multiples of 5 print `Buzz`, multiples of 15 print `FizzBuzz`, other numbers print themselves.
- [ ] A missing or non-numeric N exits non-zero with a usage line on stderr.
- [ ] `fizzbuzz --help` (or `-h`) prints usage and exits 0.
