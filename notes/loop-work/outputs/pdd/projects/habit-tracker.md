# habit-tracker

**Goal:** Mark/unmark habits per day and render a streak calendar.

**Kind:** greenfield

## Acceptance criteria
- [ ] Records per-day habit completion, computes streaks, and renders an ASCII calendar.
- [ ] State persists across invocations.
- [ ] Matches golden-file output for fixture inputs.
- [ ] Unicode/emoji content renders without column misalignment.
- [ ] Empty input produces empty output and exits 0.
- [ ] Sorting is stable for equal keys.
- [ ] `--help` / `-h` prints usage and exits 0.
